# Ocean Factory

Ocean Factory — прототип игры-головоломки на Unity в жанре автоматизационной песочницы. Игрок строит экстракторы, конвейеры и ассемблеры, производит предметы по рецептам и доставляет их в центральный хаб для прохождения уровней.

## Возможности

- Сеточная система строительства
- Процедурные залежи ресурсов
- Экстракторы базовых ресурсов
- Конвейерная транспортировка предметов
- Ассемблеры с выбором рецептов
- Прогрессия уровней через хаб
- HUD целей и окно выбора рецептов

## Технологии

- Unity
- C#
- Unity UI + TextMeshPro
- ScriptableObject-конфигурации

## Управление

| Действие | Клавиша |
|---|---|
| Перемещение камеры | ПКМ + движение мыши |
| Масштаб | Колесо мыши |
| Экстрактор | `1` |
| Конвейер | `2` |
| Ассемблер | `3` |
| Удаление | `4` |
| Поворот | `R` |
| Список рецептов | `M` |
| Отмена / закрыть окно | `Esc` |
